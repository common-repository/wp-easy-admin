== Description ==

Search Admin Menus plugin allows you to access WordPress site admin menus easily by searching them.

Just use key combination ALT + x ( can be changed ) anywhere in the admin area to open Search Admin Menus popup and select or search the admin menu that you want to visit.

Use following key shortcuts to open / close Search Admin Menus popup and access menus in it.

ALT + x ( can be changed )  - Open popup
Esc - Close popup
Down Arrow Key  - Move current menu selection downward
Tab Key         - Move current menu selection downward
Up Arrow Key    - Move current menu selection upward
Right Arrow Key - Visit current selected menu
Left Arrow Key  - Visit current selected menu
Enter Key       - Visit current selected menu
Any Other Key   - Type into search field

You can use plugin settings in the admin area to customize plugin functionality.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins -> Add New -> Search` and search for Search Admin Menus).
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/wp-easy-admin.zip "Download Search Admin Menus") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Use the `Settings -> Search Admin Menus` to configure the plugin

== Frequently Asked Questions ==

= Why does the Search Admin Menus popup screen not display on my site ? =

Please make sure you have configured the plugin settings from Settings -> Search Admin Menus and also make sure yout site is not generating any JavaScript error.
